FLUID-FIRST STRATEGY

clamp() handles sizing. Breakpoints handle layout. Resize your browser to see everything scale smoothly — no jarring jumps between breakpoints.

USE CLAMP() FOR
Font sizes (all levels)
Section padding (vertical)
Grid gaps
Card internal padding
Button padding
Container side padding
Decorative element sizing
USE @MEDIA FOR
Grid column count changes
Nav → hamburger toggle
Flex-direction switches
Showing/hiding elements
That's it. Only 2 breakpoints: 1024px & 768px

Rules

DO / DON'T

DO
Use Bebas Neue for headlines only
Keep buttons at 4px radius
Use red as the sole accent color
Left-align hero content (split grid)
Use cream background for light sections
Custom cubic-bezier for all motion
Animate only transform & opacity
Use section tags before every H2
Uppercase all nav, buttons, labels
Use clamp() for all sizing & spacing
Reference --text-* and --space-* tokens
Use min-height: 100dvh (not 100vh)
DON'T
Use Inter, Roboto, or Arial
Make pill/rounded-full buttons
Add purple, blue, or gradient accents
Center hero text in a single column
Use pure white (#FFF) as page background
Use ease-in-out or linear transitions
Animate width, height, top, or left
Skip the red dash on section tags
Use sentence case for navigation
Use @media for font-size changes
Use @media for padding/gap changes
Hard-code px values for font/spacing
